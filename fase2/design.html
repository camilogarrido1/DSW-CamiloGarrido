Etiquetas semánticas

¿Qué es el etiquetado semántico y qué hace?

Con la aparición de HTML5 aparecieron numerosas etiquetas para que Google indexara los contenidos de la web. De esta manera sus arañitas entienden qué contenido tiene relevancia y cuál no aporta valor.

Etiquetas semánticas estructurales y de contenido.

hay 2 tipos de etiquetas semánticas:

  • Etiquetas estructurales

  • Etiquetas que aportan valor semántico.

Etiquetas que aportan valor semántico:

Estas son algunas de estas etiquetas.

  • strong – Para resaltar ideas importantes (no hay que abusar de ella).

  • em – Para dar «énfasis» a ideas secundarias que tengas que ver con el contenido.

  • span– Para (dentro de un párrafo) etiquetar una sola palabra y posicionarla.

  • dfn – La uso para definir a los bots una palabra concreta.
  • bloqote - para crear citas de texto
  • cite – aunque parezca lo mismo que la anterior se usa para las imagenes sobretodo.
  • ins – Para subrayar el texto a resaltar

    CSS Grid

    CSS Grid layout contiene funciones de diseño dirigidas a los desarrolladores de aplicaciones web. El CSS grid se puede utilizar para lograr muchos diseños diferentes. También se destaca por permitir dividir una página en áreas o regiones principales, por definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML.
